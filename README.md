# ESP8266-Labs

## About

This repository serves as a curated archive of my hands-on explorations with the ESP8266 NodeMCU. Each experiment or project is:

- **Numbered and organized** for easy reference
- **Documented in separate Markdown files** for clarity and convenience

Feel free to browse, use, modify, or adapt any of these examples for your own purposes — no restrictions.
